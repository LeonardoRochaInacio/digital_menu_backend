# By-menu server side

An API to connect to a database using a interfaced class for the controllers
## API Reference

#### Get all items

```http
  GET /
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `no parameter` | `no type` | just for test, it return all the menu items |
